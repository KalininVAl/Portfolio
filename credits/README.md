### Описание проекта
На основе данных кредитного отдела банка проведена предобработка данных (обработаны пропуски, аномалии, изменены типы данных, удалены дубликаты, выполнена категоризация данных). Изучено влияние семейного положения и количества детей на факт погашения кредита в срок.

### Используемые библиотеки
pandas, matplotlib
